Faster Animation

The default next theme’s animation is quite slow, however, I found out changing enable: false under motion section, the sidebar with author name, number of posts disappeared completely.

After some research online, I decided to go for this configuration.

  • async: true allows all elements to load together
  • post_block: none allows no css animation to be applied when loading an element However, the menu area above sidebar still has some css animation effects applied, I am not sure how to change it though.
